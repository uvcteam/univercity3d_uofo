Invitation Inbox
This is where you receive invitations to events that other students have created. You can respond to an event by simply accepting an invitation. (Note: you receive invitations based upon the preferences located in Personal Vault.
Browse/Search
This allows you to search all of the events that have been created by all users regardless of preference.
Create Event
This allows you to create an event and invite other students. When you create an event you can set the amount of people that you would like to attend. The minimum number of attendees is 3 (yourself and at least 2 other UniverCity members). The maximum number of participants is up to you. Until the minimum number of people respond to the event and accept the invitation the event will remain anonymous. When the minimum number of attendies specified accept their invitations the event becomes live. At this point, all attendees will be notified of each others' identities.
Manage Events
This allows you to view the events you have created or are attending as well as cancel any events you have made and withdraw from any events that you are attending.
